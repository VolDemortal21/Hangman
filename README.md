# Hangman

A simple command-line Hangman game made in Python.

## What It Does

The game randomly picks a word from a list, then the player guesses one letter at a time.  
Correct guesses reveal letters in the hidden word. Wrong guesses add to the hangman drawing.

If the player guesses the full word before running out of attempts, they win.  
If they get too many guesses wrong, the game ends.

## Requirements

- Python 3

## How to Run

Either double-click it, or put the Python file in a folder, then open a terminal in that folder and run:

```bash
py hangman.py
```

or:

```bash
python hangman.py
```

## How to Play

1. Start the game.
2. Press Enter when asked.
3. Guess one letter at a time.
4. Correct letters will appear in the word.
5. Wrong guesses will build the hangman.
6. Guess the word before the hangman is complete.
